## 📖 项目简介

茯兴硒乡项目聚焦富硒茯茶生产核心环节，针对传统发花工艺依赖经验、质量不稳定、效率低下等痛点，研发出集 AI 视觉识别、5G 物联网和红外光谱技术于一体的智能发花设备，实现茯茶生产过程的精准控制与数字化管理。

### 核心技术
- 🤖 AI 视觉识别系统，分级精度超 98%
- 📡 红外光谱模块，实时监测硒含量与发酵状态
- 🌐 5G 物联网监测大屏，远程实时监控
- ♻️ 茶叶梯级利用与处置一体化系统
- 🔧 "四管"服务模式：管技术 + 管质量 + 管维修 + 管文创

## 🚀 快速部署到 Netlify

### 方式一：通过 GitHub + Netlify 官网（推荐）

1. **创建 GitHub 仓库**
   - 访问 [github.com](https://github.com/) 创建新仓库
   - 仓库名：`fuyichangan-promo`

2. **上传文件**
   将以下文件上传到仓库：
   - `index.html`
   - `logo.png`
   - `home.png`
   - `智能化发花装置2.0.jpg`
   - `netlify.toml`（Netlify 配置文件，可选）

3. **部署到 Netlify**
   - 访问 [netlify.com](https://netlify.com/)
   - 使用 GitHub 账号登录
   - 导入仓库并点击 Deploy

4. **访问网站**
   - 网站地址：`https://fuyichangan-promo.netlify.app`
