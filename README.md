## 🚀 快速部署到 Vercel

### 方式一：通过 GitHub + Vercel 官网（推荐）

**这是最简单的方式，无需任何技术基础！**

1. **创建 GitHub 账号**
   - 访问 [github.com](https://github.com/) 注册账号

2. **创建新仓库**
   - 在 GitHub 点击右上角 "+" → "New repository"
   - 仓库名：`fuyichangan-promo`
   - 点击 "Create repository"

3. **上传文件**
   - 点击 "uploading an existing file"
   - 拖拽以下文件到上传区域：
     - `new-promo.html`
     - `logo.png`
     - `home.png`
     - `智能化发花装置 2.0.jpg`
     - `vercel.json`
   - 点击 "Commit changes"

4. **部署到 Vercel**
   - 访问 [vercel.com](https://vercel.com/)
   - 使用 GitHub 账号登录
   - 点击 "Add New..." → "Project"
   - 选择您的仓库 `fuyichangan-promo`
   - 点击 "Deploy"
   - 等待 1-2 分钟，完成！

5. **访问网站**
   - 您的网站地址：`https://fuyichangan-promo.vercel.app`

### 方式二：使用部署脚本（适合有开发经验的用户）

双击运行 `deploy-to-vercel.bat`，按照提示完成部署。

**前提条件：**
- Node.js 已安装
- Git 已安装
- Vercel 账号

## ❓ 常见问题

### Q: 部署后页面显示 404？
**A:** 确保 `new-promo.html` 和 `vercel.json` 文件已正确上传。

### Q: 图片无法加载？
**A:** 检查文件名是否正确（包括大小写），确保图片文件已上传。

### Q: 如何绑定自己的域名？
**A:** 在 Vercel 项目设置中，进入 "Domains" 页面，添加您的域名并按提示配置 DNS。

### Q: 网站访问速度慢？
**A:** Vercel 使用全球 CDN，首次访问可能稍慢，后续会自动缓存加速。
